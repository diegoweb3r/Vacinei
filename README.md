<h1 align="center"> 🛡️ VACINEI - Carteira de Vacinação Digital </h1>

<h2 align="center"> 📖 Descrição   </h2>
<p align="justify"> O <strong>VACINEI</strong> é uma aplicação web desenvolvida para modernizar a forma como cidadãos interagem com suas informações de saúde. O projeto foca em acessibilidade e segurança, permitindo o registro detalhado de doses, lotes e locais de aplicação, além de oferecer um sistema visual de lembretes para reforços pendentes.
</p>
<p align="justify">
Este projeto faz parte da minha jornada de especialização em tecnologia, unindo conceitos de design responsivo com uma arquitetura de CSS modular voltada para a escalabilidade e manutenção de longo prazo.
</p>

---

### 🛠 Tecnologias Utilizadas  
- **Estruturação:** HTML5 Semântico para acessibilidade e SEO.
- **Estilização:** CSS3 Moderno utilizando Metodologia Modular e Custom Properties (Variáveis CSS).
- **Layout:** CSS Grid e Flexbox para responsividade intrínseca (sem dependência excessiva de Media Queries).
- **Tipografia:** Plus Jakarta Sans para uma interface contemporânea e legível.
- **Versionamento:** Git & GitHub para controle de fluxo e documentação.

### 🗣️ Comentários
<p align="justify">O VACINEI nasceu de uma necessidade real: a falta de um controle centralizado e digital que me levou a tomar duas doses repetidas de uma mesma vacina por puro erro de registro físico. Ao perceber que não havia uma solução digital acessível e intuitiva, decidi aplicar meus estudos de Pós-Graduação em IA e UX e a formação em Analise e  Desenvolvimento de sistemas para construir esta plataforma. 
O projeto está sendo desenvolvido com foco em boas práticas e escalabilidade. Utilizo a IA como meu 'Tech Lead' para garantir que cada linha de código siga padrões de mercado, preparando a base para a futura integração com bancos de dados e tecnologias mais robustas</p>
